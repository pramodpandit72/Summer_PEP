# Graph Problems

## Easy
- Number of Islands (LC #200) - BFS/DFS
- Flood Fill (LC #733) - DFS/BFS
- Clone Graph (LC #133) - BFS/DFS with HashMap

## Medium
- Course Schedule (LC #207) - Topological Sort (BFS)
- Topological Sort (LC #210) - Kahn's Algorithm
- Pacific Atlantic Water Flow (LC #417) - Multi-source BFS
- Number of Connected Components (LC #323) - Union-Find/DFS

## Hard
- Word Ladder (LC #127) - BFS Shortest Path
- Word Ladder II (LC #126) - BFS + Backtracking
- Minimum Cost to Connect All Cities (LC #1584) - MST
- Critical Connections (LC #1192) - Bridges in Graph

## Key Concepts
- BFS/DFS traversal
- Topological sorting
- Shortest path (Dijkstra, BFS)
- Union-Find
- Bridges & Articulation Points
