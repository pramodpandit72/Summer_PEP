# Tree Problems

## Easy
- Inorder Traversal (LC #94) - DFS
- Postorder Traversal (LC #145) - DFS
- Level Order Traversal (LC #102) - BFS
- Max Depth (LC #104) - Recursion

## Medium
- Zigzag Level Order (LC #103) - BFS with Direction
- Path Sum (LC #112) - DFS with Tracking
- Construct BT from Preorder & Inorder (LC #105) - Recursion
- Vertical Order Traversal (LC #987) - DFS/BFS

## Hard
- Serialize & Deserialize BT (LC #297) - BFS/DFS
- Binary Tree Maximum Path Sum (LC #124) - DFS with Returns
- Recover Binary Search Tree (LC #99) - DFS Traversal

## Key Concepts
- DFS (Inorder, Preorder, Postorder)
- BFS (Level Order)
- Tree reconstruction
- Path tracking
