For a **TCS coding round** (especially **TCS NQT / Prime / Digital style rounds**), the pattern is usually:

* Easy → Medium DSA
* Arrays, strings, hashing
* Basic greedy
* Sliding window
* Binary search
* Recursion
* Sometimes basic DP
* Implementation + edge cases matter more than advanced CP

Since you're already doing **LeetCode + C++ + Full Stack**, I’d practice these.
