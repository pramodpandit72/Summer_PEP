### 1. What is an API?

API (Application Programming Interface) is a way for two software applications to communicate with each other. It sends requests and receives responses, allowing applications to share services and data.

---

### 2. What is Gemini API?

Gemini API is Google’s AI service that allows developers to use Gemini models in their applications. It helps generate answers, text, and AI responses without building an AI model from scratch.

---

### 3. What is an LLM?

LLM (Large Language Model) is an AI model trained on a large amount of text data to understand and generate human language. It can answer questions, write text, and assist users intelligently.

---

### 4. What is Prompt Engineering?

Prompt Engineering is the process of writing clear instructions for an AI model to get better and more accurate responses. Good prompts improve the quality of AI output.

---

### 5. How does your chatbot answer resume-based questions?

My chatbot stores resume information in a prompt and sends it to Gemini API along with the user’s question. Gemini reads that information and generates answers based only on the provided resume data.

---

### 6. What is the difference between Fine-Tuning and RAG?

Fine-Tuning means retraining a model using new data so it learns permanently. RAG retrieves information from documents or databases and gives it to the model without changing the model itself.

---

### 7. Why did you use Gemini instead of training your own model?

I used Gemini because training an AI model requires a lot of data, computing power, time, and cost. Gemini provides a ready-to-use AI solution that is easy to integrate into projects.

---

### 8. How can this chatbot be improved for production use?

This chatbot can be improved by storing data in a database, securing API keys, adding login functionality, supporting PDF uploads, and using RAG for better information retrieval.

---

### 9. Is your project working successfully? What have you learned from this project?

Yes, my project is working successfully and can answer resume-based questions using Gemini API. Through this project, I learned about APIs, LLMs, prompt engineering, chatbot development, and AI integration using Python.
