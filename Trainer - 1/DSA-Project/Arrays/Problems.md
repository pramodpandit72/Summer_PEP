# Array Problems

- Two Sum (LC #1) - HashMap
- Best Time to Buy Stock (LC #121) - Single Pass
- Contains Duplicate (LC #217) - HashSet

## Medium
- 3Sum (LC #15) - Two Pointers + Sorting

- Next Permutation (LC #31) - In-place
- Product of Array Except Self (LC #238) - Prefix/Suffix
- First Missing Positive (LC #41) - In-place Hashing

## Hard
- Median of Two Sorted Arrays (LC #4) - Binary Search
- Trapping Rain Water (LC #42) - Two Pointers/Stack
- Jump Game II (LC #45) - Greedy

## Key Concepts
- Prefix/Suffix sums
- Two-pointer technique
- Sliding window
- HashMap for frequency tracking
