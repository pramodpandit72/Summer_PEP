# Heap Problems

## Easy
- Kth Largest Element (LC #215) - Min Heap
- Last Stone Weight (LC #1046) - Max Heap

## Medium
- Top K Frequent Elements (LC #347) - Min Heap
- Kth Largest in Stream (LC #703) - Min Heap (KthLargest class)
- Find K Closest Points (LC #973) - Min Heap
- Reorganize String (LC #767) - Max Heap

## Hard
- Merge K Sorted Lists (LC #23) - Min Heap of Pointers
- Find Median from Data Stream (LC #295) - Two Heaps
- Sliding Window Median (LC #480) - Heaps with Balance

## Key Concepts
- Min/Max heap usage
- Priority queue operations
- Heap construction in O(n)
- Comparison functions
